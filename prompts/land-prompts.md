# LAND Prompts — AI Prompt Library

## For Delivery Analysis

```
Here is my script for spoken delivery: [SCRIPT]

Analyse it for vocal delivery. Suggest:

1. 3 moments where I should SLOW DOWN for emphasis
2. 2 moments where I should use SILENCE (pause for 2+ seconds)
3. 2 moments where I should vary PITCH for emotional impact
4. One vocal warm-up from the HAIL framework that would help me prepare

Format your response as an annotated script with my original text and your delivery notes in [BRACKETS].
```

## For Call-to-Action Strengthening

```
Here is my story's ending: [ENDING]

My call to action is: [CTA]

Evaluate my CTA:
1. Is it specific? (Not "think about this" but "do X by Friday")
2. Is it memorable? (Could someone repeat it verbatim?)
3. Is it worth sharing? (Would someone forward it to a friend?)
4. Does it match the emotional note of the story?

Suggest 3 alternative CTAs, each targeting a different emotional trigger: urgency, inspiration, or curiosity.
```

## For Presentation Practise Feedback

```
I'm going to deliver a [LENGTH] presentation on [TOPIC] to [AUDIENCE].

Here is my preparation plan:
- I've rehearsed [NUMBER] times
- My main concern is [CONCERN]
- My previous feedback was [PREVIOUS FEEDBACK]

Suggest:
1. One specific exercise to address my concern (vocal, physical, or mental)
2. A pre-presentation ritual (5 minutes before I go on stage)
3. A "safety line" — one sentence I can return to if I lose my train of thought
```

## For Conversation Skills

```
I want to improve my conversation skills for [CONTEXT].

Give me:
1. One open-ended question I can ask to deepen the conversation
2. One listening technique to practise this week
3. One "deadly sin" to avoid and why

Start with Celeste Headlee's framework: most people listen at 225 wpm but think at 500 wpm — the gap is where distraction lives.
```
