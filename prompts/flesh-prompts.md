# FLESH Prompts — AI Prompt Library

## For Expanding an Outline into Narrative

```
Here is my three-act outline: [OUTLINE]

Help me expand each act into a full narrative. For each act, suggest:
1) A metaphor that makes the central idea concrete
2) One sensory detail (what does it look, sound, feel, smell, or taste like?)
3) A moment where the hero shows vulnerability
4) A "two plus two" moment where the audience fills in the gap

Write in a voice that sounds like [ME]. Keep total length under [WORD COUNT] words.
```


## For Data Storytelling

```
Here is my story idea about [TOPIC].

I have these data points: [DATA]

Help me weave the data into a narrative:
1. What's the human story behind the numbers?
2. What's the best visual format for each data point (line graph, bar chart, table, etc.)?
3. How do I transition between data points without losing narrative momentum?
4. What's the ONE number the audience MUST remember?

Follow Cole's rule: Put the primary takeaway IN THE TITLE.
```

## For Metaphor Generation

```
Here is my story concept: [CONCEPT]

My audience knows about: [AUDIENCE FAMILIAR DOMAIN]

Generate 3 metaphors that connect [CONCEPT] to [AUDIENCE FAMILIAR DOMAIN].

For each metaphor:
1. State the metaphor
2. Explain why it works — what shared structure do [CONCEPT] and [DOMAIN] have?
3. Test: Does the metaphor oversimplify or mislead?

Example: "CRISPR is a word processor for DNA" works because both (a) edit existing content and (b) require precision.
```

## For Production Format Guidance

```
Here is my story: [STORY]

My audience is [AUDIENCE DESCRIPTION]. My time/resource constraints: [CONSTRAINTS].

Help me decide the best production format:
1. Text (essay, blog post, email)
2. Audio (podcast, voice note)
3. Slides (presentation, lecture)
4. Video (social, teaching)
5. Storyboard (pre-production planning)

For the recommended format, provide one specific template or outline I can follow.
```

## For Bilingual Story Adaptation

```
Here is a story written in [SOURCE LANGUAGE]: [STORY]

Help me adapt it for [TARGET LANGUAGE].

CRITICAL: Do NOT translate literally. Re-craft the story.

Consider:
1. Which metaphors need replacement?
2. Which cultural references need explanation or substitution?
3. Does the pacing change? (Some languages need more/fewer words to convey the same emotion.)
4. Does the vulnerability read differently in [TARGET LANGUAGE]?

Output: The re-crafted story in [TARGET LANGUAGE] + a 3-sentence explanation of the key adaptations made.
```
