# HOOK Prompts — AI Prompt Library

## For Generating Hook Angles

```
Here is my story idea: [DESCRIBE YOUR STORY]

Generate 3 different hook angles for this story. For each hook:
1. Write the hook itself (1 sentence)
2. Explain who this hook would resonate with most
3. Name the hook type (Promise / Gap / Wonder)
4. Rate the hook's memorability out of 10
```

## For Testing Hook Effectiveness

```
Here are 3 hooks I'm considering for my story:

[Hook 1]
[Hook 2]
[Hook 3]

For each hook, predict:
1. What question would the audience have after hearing this hook?
2. What emotion does this hook create?
3. Would this hook work better in English, Cantonese, or Mandarin?
4. Which hook is strongest and why?
```

## For Bilingual Hook Adaptation

```
Here is a hook that works in English: [HOOK]

Help me adapt this hook for a Cantonese-speaking audience and a Mandarin-speaking audience.

For each language:
1. Rewrite the hook (do NOT translate literally — re-craft)
2. Explain how the cultural reference or emotional trigger changes
3. Rate the hook's likely effectiveness on a scale of 1-10 in that language
```

## For Weak Hook Rescue

```
Here is my current hook: [HOOK]

It doesn't seem to be working. Help me diagnose the problem:
1. Is it too vague? (No specific image or outcome)
2. Is it too complex? (More than one idea)
3. Is it too safe? (No tension or gap)
4. Is it too familiar? (Audience has heard it before)

Then suggest 3 alternative hooks that fix the problem.
```
