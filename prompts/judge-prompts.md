# JUDGE Prompts — AI Prompt Library

## For Full Story Critique

```
Here is my story: [STORY]

Act as a constructive critic. Score my story against these criteria:

| Criterion | Score /10 | Reason | One Suggestion |
|-----------|-----------|--------|----------------|
| Clarity — Is the single idea clear? | | | |
| Emotion — Does it create a feeling? | | | |
| Structure — Is the three-act arc visible? | | | |
| Originality — Does it feel fresh? | | | |
| Authenticity — Does it feel genuine? | | | |

Also answer:
1. In one sentence, what is this story about?
2. Who is this story FOR?
3. What's the weakest moment and why?
4. If I had to cut 50%, what would you keep?

Be specific. Say "On page 2, the transition from X to Y feels rushed because..." not just "The pacing is off."
```

## For the Single-Idea Test

```
Here is my story: [STORY]

I want to verify that ONE clear idea runs through everything. 

1. State the single idea of this story in ONE sentence.
2. List every section/scene that supports this idea.
3. List every section/scene that distracts from this idea.
4. For each distracting element, suggest: cut, simplify, or reframe to connect to the main idea.

If a reader remembers only the single idea, will the rest of the story make sense?
```

## For the Ethics Check

```
Here is my story: [STORY]

I used AI to help craft this story. Check for ethical issues:

1. Does the story feel like it has a human voice, or does it read like AI output?
2. Are there any narrative stereotypes or biases being reinforced?
3. Does the vulnerability feel authentic, or performative?
4. If an audience member knew AI was used, would they feel misled?
5. Who benefits from this story? Who might be harmed?

Rate overall ethical health: Green (safe to publish) / Yellow (needs revision) / Red (do not publish)
```

## For Audience Testing

```
Here is my story: [STORY]

My target audience is: [AUDIENCE DESCRIPTION]

Predict this audience's response:

1. What will they remember 24 hours later?
2. What questions will they ask?
3. What part will they most connect with emotionally?
4. What part will they most disagree with or question?
5. What call to action are they MOST likely to take?

If the gap between "what I want them to remember" and "what they will remember" is too large, suggest specific revisions.
```
