# Workshop Blueprint: 1-Day Storytelling Intensive

A deep dive into all 5 stages of the Storyteller Framework. Suitable for university students, educators, or professional training.

## Learning Objectives

By the end of this workshop, participants will:
- Write a complete story from hook to delivery
- Create a visual storyboard
- Give and receive structured peer feedback
- Deliver a 60-second story with vocal awareness
- Understand how AI can amplify (not replace) storytelling

## Materials

- Printed framework handouts
- Storyboard templates
- LEGO bricks (one set per 3-4 people)
- AI tool access (e.g. ChatGPT, LexStory)
- Recording device (phone)

## Agenda

### Morning Session: Foundation

| Time | Duration | Activity | Notes |
|:-----|:---------|:---------|:------|
| 9:00-9:15 | 15 min | **Welcome + Circle:** "One sentence I want remembered" — facilitator models | Start with a story |
| 9:15-9:30 | 15 min | **Warm-up:** "The Object Hook" — bring an object, invent a story hook in 60 seconds | LSP-inspired, fast, fun |
| 9:30-10:00 | 30 min | **HOOK deep dive:** Write ending first → Find the gap → 3 hook types → AI-assisted hook generation | Use hook prompts |
| 10:00-10:30 | 30 min | **SHAPE deep dive:** Three-Act Structure → Character spine → Golden Circle → "Build the Arc" exercise | Use shape prompts |
| 10:30-10:45 | 15 min | **Break** | |
| 10:45-11:45 | 60 min | **FLESH deep dive:** Outline → Script → Metaphor → Sensory detail → "Build the Emotion" warm-up | Use flesh prompts |
| 11:45-12:30 | 45 min | **Production workshop:** Storyboard creation (6-8 frames). Draft assets. | Storyboard template |
| 12:30-13:30 | 60 min | **Lunch** | |

### Afternoon Session: Polish & Present

| Time | Duration | Activity | Notes |
|:-----|:---------|:---------|:------|
| 13:30-14:00 | 30 min | **JUDGE deep dive:** Self-check → Peer feedback (Three Loves / One Suggestion) → AI-assisted critique | Use judge prompts |
| 14:00-14:30 | 30 min | **Revision sprint:** Apply feedback. Strengthen the weak moment. | |
| 14:30-14:45 | 15 min | **"The Blind Critique"** — LSP warm-up: listeners build emotional shape of the story | Powerful feedback exercise |
| 14:45-15:15 | 30 min | **LAND deep dive:** Vocal toolbox → HAIL warm-up → Celeste Headlee's 10 rules | Do all 6 vocal exercises |
| 15:15-15:30 | 15 min | **Break + preparation** | |
| 15:30-16:15 | 45 min | **Story showcase:** Each person delivers 3-minute story. Audience gives One Question each. | Record if possible |
| 16:15-16:45 | 30 min | **Reflection + Closing:** What did you learn? What will you do differently? | |
| 16:45-17:00 | 15 min | **Resources + Next steps:** How to use the AI prompts, where to find templates | |
