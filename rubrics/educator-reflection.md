# Educator Reflection — Self-Assessment for Trainers

This is not a grading rubric — it's a reflection tool for educators who teach storytelling. Use it to evaluate your own facilitation.

## Before the Workshop

| Question | Notes |
|:---------|:------|

| Which Storyteller stages will I cover? | |
| What's my single goal for this session? | |
| How will I adapt for this group's language/culture? | |
| What AI tools will I use or demonstrate? | |
